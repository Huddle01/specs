In this document, we will introduce the Huddle01 Protocol, a decentralized, peer-to-peer, real-time communication protocol that enables secure, private, and scalable communication between users. (THIS IS A PLACEHOLDER)

### Orchestrator Nodes
Orchestrators select the best media servers and coordinates their interaction. It also maintains websocket connections with clients for WebRTC signaling and messaging.

### Media Nodes
is an abstraction of WebRTC SFU (Selective Forwarding Unit) which processes and manages media streams , ensuring media communication between clients

### Registry Nodes
keeps track of all the media nodes which are available for use , monitors health and reports back on any abnormalities
